### GitHub Trending 热点项目

1. **AIRI - 自托管 AI 伴侣** (21,221 stars)
   - moeru-ai/airi 是一个自托管的 AI 伴侣项目，目标是复刻 Neuro-sama 的高度。支持实时语音聊天、Minecraft 和 Factorio 游戏，跨平台支持 Web/macOS/Windows。**这是 AI 虚拟角色领域的又一重磅项目，让每个人都能拥有自己的 AI 伴侣。**

2. **ruflo - Claude Agent 编排平台** (17,837 stars)
   - ruvnet/ruflo 是领先的 Claude Agent 编排平台，支持部署多 Agent  swarm、编排自主工作流、构建对话式 AI 系统，具备企业级架构、分布式 swarm 智能和 RAG 集成。**标志着 Claude 生态正在向规模化自治方向演进。**

3. **Alibaba OpenSandbox** (3,946 stars)
   - 阿里巴巴开源的通用沙箱平台，为 AI 应用提供多语言 SDK、统一的沙箱 API，支持 Docker/Kubernetes 运行时，适用于 Coding Agents、GUI Agents、Agent 评估等场景。**阿里在 AI 基础设施领域的开源贡献值得关注。**

4. **Claude Scientific Skills** (10,872 stars)
   - K-Dense-AI/claude-scientific-skills 提供 148+ 即用型科学研究技能，涵盖生物信息学、化学信息学、蛋白质组学、临床研究、医疗 AI 等领域。**让 AI Agent 真正具备科学研究能力的重要一步。**

5. **Superset - AI Agents 的 IDE** (3,320 stars)
   - superset-sh/superset 是 AI Agents 时代的终端 IDE，支持在本地同时运行多个 Claude Code、Codex 等 Agent，每个任务隔离在独立 git worktree 中。**多 Agent 并行协作的开发范式正在走向成熟。**
